Weeks 1-6: Foundations of front end programming.

In this time frame you will do one pass over all the main skills you need in order to be a front-end programmer. You will learn HTML, CSS, Bootstrap, JavaScript, jQuery, Git, Ajax programming, and build tools (Browserify, npm and Gulp)

Weeks 7-15: Advanced Front End Programming

Our faculty of real industry pros takes you through more advanced front-end development techniques in an environment of project-based learning. You will learn techniques for using gradients, HTML5 media, HTML5 canvas, CSS transforms and perspective, and lots of event-driven and AJAX programming.

Weeks 16-21: MongoDB and Node

You will build at least 6 full stack applications using NodeJS and MongoDB or SQL.

Weeks 22-36: AngularJS, php and Beyond

During this last stretch, you will learn AngularJS, ReactJS, php, and become a WordPress pro. We're not talking about clicking around a gui and installing themes, we're talking about harnessing the WordPress codex to build real database-driven apps on the back of WordPress. You will also learn more advanced git techniques and brush up on Computer Science fundamentals
